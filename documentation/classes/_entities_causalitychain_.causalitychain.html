Class CausalityChain<S, T>

A causality chain is a complex interconnection of links. Each link has a number of incoming links and a number of outgoing links. The chain keeps track of causal relations between events of type T. By keeping track of these causal relationships, loops (i.e. events causing themselves to reoccur, thereby potentially creating an infinite loop of recurrence) can be detected and summarily avoided.

Adding new events to the causal chain occurs in two phases. First, we try to introduce the event into the staging area. This is where we check for potential periodicity. After a given round (a time step in the causal chain), all staged events are integrated into the chain. At this point we also check whether there has been a break in causality for any of the events at the tail end of the chain. If so, then all paths leading (unidirectionally) to these events must be severed.
